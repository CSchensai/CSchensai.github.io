# dumi library

## Getting Started

Install dependencies,

```bash
$ npm i
```

Start the dev server,

```bash
$ npm start
```

Build documentation,

```bash
$ npm run docs:build
```

Build library via `father-build`,

```bash
$ npm run build
```

```
// 需要10.13.0以上
"gitHooks": {
    "pre-commit": "lint-staged"
  },
```
